  • 01

에이전트 자동 실행 방식 자세히 보기

CLI 기반 AI 에이전트는 명령어 추천을 넘어 직접 실행하고 결과를 확인합니다.

작동 흐름

  1. 사용자가 목표를 말합니다.
  2. AI가 작업 순서를 판단합니다.
  3. 파일 읽기, 명령어 실행, 코드 수정을 수행합니다.
  4. 결과를 확인하고 다시 수정합니다.
